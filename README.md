This is a premade automod script that anyone can use for their subreddit with little modification. 
<br>Due note that this will need to be adapted to your specific use case as this is designed for a Murder Drones community subreddit.

It features the following:
* Pinned automod comment under posts
* Filters accounts younger than a week
* Filters accounts with less than 20 karma
* Filters reported posts
* Filters non-English/Spanish/Portuguese letters
* Filters blacklisted subreddits
* Filters potental slurs/political content
* Filters pornographic domains
* Filters doxxing (Phone numbers/Email addresses/Card numbers/IP addresses/Street addresses)
  
  
NOTE: This only filters the offending content, not auto-removes. This will add the content to the mod queue for human review before removal.
